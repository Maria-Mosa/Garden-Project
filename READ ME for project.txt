## בשביל להוריד ולהריץ את הפרויקט צריך להוריד :
1- Visual Studio Code
2- bootstrap

## הפרויקט נמצא תחת תיקיית ptfy
O(:C) / משתמשים /PTFY /USER   

## בתוך הפרוייקט תחת תיקיית src\app יש כמה components :

*1* component canvas : 
הדף המיוצג אחרי לחיצה על כפתור  Draw
המשתמש יכול לשרטט את הגינה שלו .

*2*component login-page : 
הדף הראשי שיוצג באתר , שכולל תקציר על האתר ולבחור סוג השיבוץ .

*3*component rest-data :  
הדף המיוצג אחרי לחיצה על כפתור  Go To Complete Data
המשתמש ימשיך במילוי הנתונים , ואחרי לחיצה על GO יופק הפתרון האופטימלי.

**4 component shape-form :
המרובע בתחתית דף השרטוט שמייצג את הממדים . 

**5 component shape-type :
ה combobox שנמצא בדף השרטוט שיש בו סוגי הצורות ועוד combobox לבחירת המשתמש meter/cm
  
*6* shape : 
השדות שיש לכל צורה שאפשר שישרטט אותה המשתמש.

## בתוך הפרוייקט תחת תיקיית src\assests יש תיקיית images שכוללת כל התמונות שהשתמשנו בפריוקט. 

## בשביל להריץ הפרויקט צריך לפתוח new terminal ולהריץ הפקודה הבאה : ng serve --open
